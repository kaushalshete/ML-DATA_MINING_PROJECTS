# 📊 Knowledge Discovery & Machine Learning Projects

This repository contains **11 hands-on projects** covering key concepts in **Data Preprocessing, Feature Engineering, Statistics, and Machine Learning**. Each notebook demonstrates a specific technique used in real-world data science workflows.

---

## 🚀 Projects Overview

### 1. 📌 KDD (IQR_TEST)
Implements **Interquartile Range (IQR)** method to detect and handle outliers in datasets.

---

### 2. 📌 KDD (Z_SCORE)
Uses **Z-Score technique** for identifying outliers based on standard deviation.

---

### 3. 📌 KDDBINNING
Applies **data binning techniques** (equal-width / equal-frequency) for data smoothing and discretization.

---

### 4. 📌 KDDDATAPREPROCESSING
Covers **data cleaning steps** including handling missing values, duplicates, and basic transformations.

---

### 5. 📌 KDDENCODER
Demonstrates **categorical data encoding** methods like Label Encoding and One-Hot Encoding.

---

### 6. 📌 KDDKNNCLASSIFICATION
Implements **K-Nearest Neighbors (KNN)** algorithm for classification tasks.

---

### 7. 📌 KDDOUTLIERHANDLING
Shows multiple techniques to **detect and treat outliers** in datasets.

---

### 8. 📌 KDDREGRESSION
Applies **regression models** to predict continuous values and analyze relationships.

---

### 9. 📌 MLProject01
A complete **end-to-end ML pipeline project** including preprocessing, training, and evaluation.

---

### 10. 📌 MLProject02
Another applied ML project focusing on **model improvement and performance optimization**.

---

### 11. 📌 Sonar Rock vs Mine
Classification project using the Sonar dataset to distinguish between rocks and mines using machine learning models.

---
